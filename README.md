# form-frontend-service

- Front end for the client facing car inquiry form.

## Local Testing

- Launch HTML page using live server or browser
- Change `const API_BASE= "/api/" to const API_BASE = "http://localhost:7071/api"` assuming local function is running on port 7071

## Notes

- Function Backend must be running to retrieve vehicles and submit form
- Local CosmosDB emulator must be running to retrieve vehicles
